Basics

Data.Tables is a powerful library for data preperation. The main goal is to make the process overall faster and use the resources more efficiently than similar libraries like dplyr or python.pandas.

Advantages

  • Compact code
  • Speed -> Fread, Fwrite, Joins,
  • Memory efficiency -> no duplicates

Disadvantages

  • Unintelligible syntax -> partly because of the compact code.
  • (No duplicates)

Benchmark

Syntax

Basic

DT[ i, j, by ]

  • i -> on which rows?
  • j -> what to do?
  • by -> grouped by what?

Chaining

DT[i, j, by][i, j, by]…n+1

Special code

  • .sd -> Select the columns for the operation
  • .() = c() -> List
  • .N -> Number of rows in the group
  • .GRP -> Number of the group
